GUI游戏界面

GUI高级控件

1.Label控件

作用：可以在游戏界面中以文本的形式展示出一段字符串信息。

注意事项：编码格式修改为（utf-8），否则中文显示不出来。

方法：Start()方法，只执行一次，需将所有初始化的相关代码都放在此方法内。

代码示例：

//接受外部赋值字符串
var str : String;
//接受外部赋值贴图
var imageTexture : Texture;
//贴图宽度
private var imageWidth : int;
//贴图高度
private var imageHeight : int;
//当前屏幕宽度
private var screenWidth : int;
//当前屏幕高度
private var screenHeight : int;

fuction Start()
{
    //得到屏幕宽高
    screenWidth = Screen.width;
    screenHeight = Screen.height;
    //得到图片宽高
    imageWidth = imageTexture.width;
    imageHeight = imageTexture.height;
}

fuction OnGUI()
{
    //将文字内容显示在屏幕中
    GUI.Label(Rect(100,10,100,30),str);
    GUI.Label(Rect(100,40,100,30),"当前屏幕高：" + screenHeight);
    GUI.Label(Rect(100,80,100,30),"当前屏幕宽：" + screenWidth);
    //将贴图显示在屏幕中
    GUI.Label(Rect(100,120,imageWidth,imageHeight),imageTexture);
}

3.本地观看效果：hexo s;

4.markdown标题格式后面加个空格，格式如“### 你好”。否则hexo辨别不出

5.代码块的部分需要缩进4个空格或是一个制表符。

自学unity（一）

project视图

作用：主要存放游戏中所需要用到的资源文件。包括游戏脚本，预设，材质，自定义字体等。这些可以赋值给游戏对象。

hierarchy视图

作用：主要存放游戏场景中具体的游戏对象，比如摄像机，平面贴图，3d贴图，光源等。

inspector视图

作用：可以展示对象的具体信息，比如说位置什么的，存放游戏对象，游戏资源，游戏设置以及描述信息的地方。

scene视图

  1. 作用：存放游戏中的模型资源。比如主角，敌人等。

  2. 某些功能：移动模型

    function Update()
{ 
//移动模型
transform.Translate(Vector3.forward * Time.deltaTime);
}

transform.Translate()方法用于设置游戏对象平移的方向。Vector3.forward表示移动的方向为前方，Time.deltaTime表示Update()方法上一帧持续的时间，这两个的乘机就是前方模型一次移动的距离。

此外还有旋转模型，缩放模型等。

game视图

作用：可在此视图中运行游戏，不过其显示的内容完全取决于摄像机所照射的部分。

导出与导入

导出：选择Assets->Export Package菜单项。

导入：直接双击“.unitypackage”游戏工程包。

第一个游戏实例

代码脚本如下，其中解释：OnGUI()用于绘制GUI界面组件。Rect表示规定一个矩形区域，用于显示控件。Vector3标志一个模型旋转移动的方向。transform.Rotate()设置模型旋转，transform.Translate()设置模型平移。GUI.Button()设置一个按钮，返回true时按钮被按下。GUI.Label()设置一个文本框。

#pragma strict
//模型移动速度
var TranslateSpeed = 20;
//模型旋转速度
var RotateSpeed = 1000;
//绘制UI
function OnGUI () {
//设置GUI背景颜色
GUI.backgroundColor = Color.red;
if(GUI.Button(Rect(10,10,70,30), "向左旋转"))
{
 //向左旋转模型
 transform.Rotate(Vector3.up * Time.deltaTime * (-RotateSpeed));
}

if(GUI.Button(Rect(90,10,70,30), "向前移动"))
{
 //向前移动模型
 transform.Translate(Vector3.forward * Time.deltaTime * TranslateSpeed);
}
if(GUI.Button(Rect(170,10,70,30), "向右旋转"))
{
 //向右旋转模型
 transform.Rotate(Vector3.up * Time.deltaTime * (RotateSpeed));
}
if(GUI.Button(Rect(90,50,70,30), "向后移动"))
{
 //向后移动模型
 transform.Rotate(Vector3.forward * Time.deltaTime * (-TranslateSpeed));
}
if(GUI.Button(Rect(10,50,70,30), "向左移动"))
{
 //向左移动模型
 transform.Translate(Vector3.right * Time.deltaTime * (-TranslateSpeed));
}
if(GUI.Button(Rect(170,50,70,30), "向右移动"))
{
 //向右移动模型
 transform.Translate(Vector3.right * Time.deltaTime * (TranslateSpeed));
}

//显示模型位置信息
GUI.Label(Rect(250,10,200,30),"模型的位置" + transform.position);
//显示模型旋转信息
GUI.Label(Rect(250,50,200,30),"模型的旋转" + transform.rotation);
}

最后为其中一个模型设置一个刚体(Rigidbody)属性。从Component到Physics到Rigidbody。

至此，一个游戏实例就完成了。

读《偷影子的人》有感

《偷影子的人》是由法国的作家马克’李维写的一部令整个法国为之动容的温情疗愈小说。记得买的书上这样描述这一本书，法国媒体惊叹：销售得比影子消失的速度还快！小心偷影子的人，他会带走你的心。

这本书讲的是一个老受班上同学欺负的瘦弱小男孩，因为拥有一种特殊能力而强大：他能“偷别人的影子”，因而能看见他人心事，听见人们心中不愿意说出的秘密。他开始成为需要帮助者的心灵伙伴，为每个偷来的影子找到点亮生命的小小光芒。在这部书里，作者写了伊凡，克蕾儿，吕克等其他人。这些人都是主角通过影子的交流知道了他们心中的秘密，同时主角也尽了最大的努力去帮助这些人，帮助他们去直视自己内心的思恋，去面对自己的不足，去实现自己的梦想。

本文中描述了主角与吕克，伊凡之间的友谊，与母亲之间的亲情，与克蕾儿，苏菲之间的爱情。整部小说只因为以主角特有的能力将这些种种表现出来，

随笔

之所以起这个名字，是因为虽然看完了《数学之美》但我却并不了解作者想要传达给我们的意思。所以，这算不上是一篇读后感，只能是我自己的一些想法（可能有些并不是书中所传导给我的）。

《数学之美》，最初看这本书的兴趣是因为第一：他与数学有关；第二：可以装X。但是，当我看完了这本书，我发现并不是我所想的那样。

简单！简单！简单！作者在这本书中多次写到简单是最美的!确实，我在高中学数学的时候最喜欢那些题目简单的题目（尽管老师说这种题一般都很难）。除了作者说的简单之美之外，在这本书中，作者用一种非计算机专业能看懂的语言向我们展示了多种算法，极尽算法之美。就目前我知识的水平来说，这本书最大的作用就是让我开拓了视野，丰富了我大脑的储备知识。虽然，里面的很多算法我都不是太了解（有些看了书还不懂，甚至用百度去了解），但是，这只是目前的状态，作为一个以后要成为程序员的人，算法什么的最终肯定要弄明白的。

对这本书第一次的描述，我就写这么多。以后如果有时间的话，我还会再读一遍，到时候再写一篇吧！

By Lirito.

看《肖申克的救赎》有感

忙着活，或忙着死。

故事的开头写到了主角安迪因为妻子偷情，想要去杀害妻子却没有实行，而之后妻子死了，自己虽然是有杀人动机的但却没有足够的证据和说服力来证明自己的清白，最终被判无期徒刑，来到了肖申克。影片到了这里，便开始引入了另一个重要人物：老囚犯瑞德。在整部影片中，导演通过瑞德的视角向我们展示了主角在肖申克的生活。在监狱的前两年，主角遭受了种种欺凌。之后，主角勇敢的为自己争取了一次机会，用自己的专业知识帮助监狱里最凶残的狱长躲过税收的麻烦。然后，全监狱的狱长，狱警都找他帮忙，而他也有了许多的自由的空间。之后，随着一个年轻的小伙子汤米的到来，主角无意知道杀害自己妻子的真正凶手，尽管安迪想要翻案，但监狱长从中作梗，秘密杀害了汤米。安迪知道自己无望出去。因此，越狱在他的头脑中形成了。于是，在一个雷电交加的夜晚，他爬过自己挖了十九年的通道，爬过五百码的化粪道，成功的离开了肖申克监狱。随后，他向记者报告了监狱贪污腐败的事，而他也重获新生。最后，他的好友瑞德也在假释后来到了他的身边。

故事十分简单，如果用一句话来说就是主角成功越狱的故事。但是，这部电影吸引我的并不是这个故事。首先，我很佩服安迪的毅力，一条通道，足足挖了十九年，如果给我十九年的时间，我相信我是挖不出来的，更何况是在监狱那种看管严的情况下。第二，我佩服安迪的耐力，在前两年的监狱时光中，安迪虽然遭受到了种种欺凌，但他并没有放弃，特别是他在帮助自己的朋友拿到了酒之后，他一口酒也没喝，但他却是一副十分享受的样子，用影片的话来说，他在享受自由。可见这是一个多么豁达的人。第三，我佩服安迪的聪明才智，在他帮监狱长偷税的那一刻，他可能已经想好以后的路了，并且，这种聪明也在他对瑞德说的话体现了出来，他叫瑞德出去后到一个地方找到最大的树，树下有他留给他的东西，而这个东西，在主角越狱之前是没有的，可见主角的目光之远。

就我看来，除了主角，我对影片中瑞德，汤米以及监狱的图书馆管理员这三个人物印象最深。其中，图书馆管理员是一位将人生的一大半都放在监狱里的人，用瑞德的话来说，他是离不开监狱的人，他已经深深的离不开此地。最后，这位老人在出狱后因为不适应生活等其他原因而上吊自杀了。汤米，一个朝气蓬勃的年轻人，他为主角带来了希望，而主角也将他视为希望，但是，因为他挡在监狱长发财的道路上，最终也被监狱长害死了。瑞德，故事的整个见证人，他与主角有着非常深厚的友谊，因为他的资格老，他在监狱也能得到他想要的任何都东西，而正因为这样，主角也经常找他买东西。最后他也得到了主角的帮助，开始了新的生活。这三个人各有各的人生，有的是完美的，有的是悲壮的。但是，这就是人生，这就是生活，命运全掌握在自己的手中，你想把他变成什么样，他就是什么样。

同时，影片也有一些十分经典的句子，如瑞德说的：这些墙很有趣。一开始你恨他，慢慢你就习惯了，到后来你就不能离开他了，这就是体制化。但是，经典归经典，虽然这句话说的有理，但我们应该向主角一样充满希望，不放弃。“希望是好的，也许是人间至善”。

By Lirito.
